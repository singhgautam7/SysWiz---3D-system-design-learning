The two shapes of scaling

When one machine can't keep up, you have two moves. Make it bigger (vertical), or add more (horizontal). Watch both racks side by side, then tap fail a node to see how each survives.

AVertical — scale up

Move to a bigger box: more CPU, more RAM, faster disks. Simple, no code changes, and every request goes to the same place. But hardware has a ceiling, and the machine is a single point of failure.

BHorizontal — scale out

Put a load balancer in front of many smaller machines. Each is disposable. Failure is absorbed instead of downtime. In exchange, services must be stateless (or externalize state).

YOUR NOTE · page 1 Jul 12
Ask Priya about the p99 curve — does it flatten because the LB is a bottleneck, or because of coordination overhead across replicas?
TRY IT
Bring NODES up from 1 → 8. Watch throughput climb while p99 flattens after ~5.

Key takeaways

  • ◆Vertical is fast; horizontal is survivable.
  • ◆Horizontal needs a load balancer + stateless services.
  • ◆Throughput scales roughly linearly; latency doesn't.

Many producers → one consumer

Three services publish independently. One worker processes. Notice how bursts from any producer are absorbed by the queue — the consumer sees a smooth 2.5/s.

Ordering, per producer

Because it's FIFO, order is preserved for messages from the same producer. Across producers, ordering isn't guaranteed — MSG 44 from payments may land after MSG 45 from orders.

Delivery: poll vs push

This diagram is on poll: worker asks the queue every 200ms. Toggle it — with push, the queue calls the worker as soon as a message lands. Push is lower latency; poll is simpler under back-pressure.

BACK-PRESSURE
Arrival is 6.2/s, drain is 2.5/s. Depth keeps growing — you either add consumers or shed load.
TRY IT
Drop the slider to 2/s — queue drains to 0. Push it to 12/s — depth explodes. Tap + add consumer and watch drain double.

Key takeaways

  • ◆Queues decouple arrival from processing.
  • ◆FIFO holds per-producer; global order is not guaranteed.
  • ◆If drain < arrival for long, depth is unbounded.
